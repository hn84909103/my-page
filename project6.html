User Manual Design

InDesign/Illustrator

This project is about to create a user manual for the users of the tablet with thermal scan. The user manual divided into two parts: physically install the tablet and the software setting.
I choose to use minimalist style to create a modern look with the blue theme color related to the company’s branding. All the illustrations of the instructions are created in Illustrator and edited in inDesign.

  • Clear instruction of connecting the camera to the tablet
  • Steps for language setting in the tablet
