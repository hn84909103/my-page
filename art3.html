3D Perspective Swing

JavaScript WebGL

To interact, you can press the keys on the keybpard:

  • Key 0: reset animation to initial state
  • Key 1: take one step
  • Key g: "go" — start the animation loop
  • Space Bar: stop the animation

There are controls for the view perpective on the top right.
